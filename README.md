## 🛑 Why SyncSpend?
Most expense trackers force you to create an account, upload your personal financial data to their cloud servers, and bombard you with ads.

We built **SyncSpend** because we believe your financial data belongs to *you*.
- **Zero Cloud:** Your data never leaves your device.
- **Zero Ads:** Clean, distraction-free interface.
- **Zero Accounts:** Open the app and start logging immediately.

## ✨ Features
* 🔒 **Absolute Privacy:** 100% local storage. We couldn't see your data even if we wanted to.
* ⚡ **Lightning Fast:** Built with Flutter for a buttery-smooth 60fps experience.
* 📊 **Clean Analytics:** Beautiful charts that make understanding your spending habits effortless.
* 🌙 **Dark Mode First:** Because we care about your retinas.

## 🛠️ Tech Stack
* **Framework:** Flutter / Dart
* **Architecture:** Offline-First

## 🚀 Building from Source
Want to audit the code or build the APK yourself? It's simple:

```bash
# 1. Clone the repository
git clone [https://github.com/a4studios04/SyncSpend.git](https://github.com/a4studios04/SyncSpend.git)

# 2. Navigate into the directory
cd SyncSpend

# 3. Get dependencies
flutter pub get

# 4. Run the app
flutter run
